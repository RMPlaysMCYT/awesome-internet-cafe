# Contribution Guidelines

Pull requests are warmly welcome. To keep the list useful:

## What to add

- Software, tools, services, hardware, tutorials, or communities **directly relevant** to operating an internet cafe, cybercafe, gaming center, or PC bang.
- Items must be in active use (or recently maintained) — not abandoned for 5+ years unless they are still de-facto standard.
- Open source projects should have a working build and a usable README.
- Commercial software is welcome if it is widely deployed in real cafes.

## What not to add

- Generic IT tools with no specific cafe relevance.
- Affiliate links or self-promotion of products you sell.
- Closed beta or pre-alpha projects.

## Format

- Add new items in alphabetical order within their category.
- One line per item: `- [Name](url) — Short description in one sentence.`
- Keep descriptions under 120 characters.
- End the description with a period.

## Pull request

- Search the existing list and open issues first to avoid duplicates.
- One item per pull request when possible.
- Title your PR `Add [Item Name]` or `Update [Item Name]`.
- In the description, briefly explain why the item belongs on this list.

Thank you for helping make this list better.
